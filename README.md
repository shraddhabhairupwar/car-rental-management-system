# 🚗 Car Rental Management System

## 📌 Overview
The Car Rental Management System is a full-stack web application developed using PHP and MySQL to manage vehicle rentals efficiently. The system allows administrators to manage cars, customers, and rental bookings through a structured dashboard interface.

## 🚀 Features
- Admin Dashboard with system statistics
- Add, Edit, Delete Vehicles
- Customer Management
- Rental Booking with Start & End Date
- Car Availability Status Tracking
- Automated Rental Record Management
- Price Per Day Calculation
- Secure Database Integration

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## 🗄️ Database
- MySQL database for storing car details, customer records, and rental transactions.
- Structured tables with relational data management.
